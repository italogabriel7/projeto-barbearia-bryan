# ✂️ Projeto Landing Page - Bryan BarberX

## 📌 Descrição do Projeto

Landing page desenvolvida para a barbearia **Bryan BarberX**, com foco em presença digital profissional, fortalecimento de marca e facilitação do contato com clientes.

O projeto foi construído a partir de um levantamento real de requisitos com o cliente, buscando atender às necessidades do negócio e aplicar conceitos de desenvolvimento web responsivo.
foi cobrado um valor de 200 R$( comrovante no video)
---

## 🎯 Objetivo

Desenvolver uma landing page profissional para apresentar:

* identidade visual da barbearia
* serviços oferecidos
* preços
* localização
* redes sociais
* contato via WhatsApp

---

## 🔎 Levantamento de Requisitos

O cliente informou:

* **Nome:** Bryan BarberX
* **Tempo de mercado:** 4 anos
* **Localização:** Dirceu, Teresina - PI
* **Público-alvo:** Jovens
* **Faixa de preço:** Popular
* **Estilo visual:** Moderno / Escuro
* **Objetivo principal:** Fortalecer a marca

---

## 💻 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* AOS (Animate On Scroll)
* Font Awesome

---

## ⚙️ Funcionalidades Implementadas

* Header com identidade visual
* Link direto para Instagram
* Hero section com chamada principal
* Botão de agendamento via WhatsApp
* Seção Sobre
* Serviços com preços
* Google Maps integrado
* Footer com créditos
* Botão flutuante de WhatsApp
* Layout responsivo
